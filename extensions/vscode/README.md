# JacLy — Jaculus + Blockly for VS Code

A VS Code extension providing a visual Blockly editor for `.jacly` files, designed for use with the [Jaculus](https://jaculus.org) JavaScript runtime for embedded devices.

## Features

- Custom editor for `.jacly` files with a Blockly visual programming interface
- Seamless integration with Jaculus projects
- Webview-based editor powered by React and Blockly

## Requirements

- VS Code 1.109.0 or later
- A Jaculus-compatible project
