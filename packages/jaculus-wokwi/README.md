# `@jaculus/wokwi`

Headless Wokwi simulator integration for Jaculus tools. The package wraps Wokwi client usage and provides React helpers for embedding simulator state in Jaculus applications.

Related Wokwi CLI changes were upstreamed in pull requests 34-37, covering the monorepo migration, client package separation, WebSocket transport, documentation, and package entry fixes.

## Development

```bash
pnpm --filter @jaculus/wokwi build
pnpm --filter @jaculus/wokwi type-check
```
